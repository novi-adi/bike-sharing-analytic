# Bike Sharing Usage Analysis

## Setup environment
```
conda create --name main-ds python=3.9
conda activate main-ds
pip install numpy pandas matplotlib seaborn scikit-learn jupyter streamlit babel
```

## Run steamlit app
```
streamlit run dashboard.py
```
